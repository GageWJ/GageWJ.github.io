Resume Challenge

I’ve set it up so that the flag you find in each challenge serves as the password to access the next one. The order of the challenges does not determne difficulty, I arranged them in a random order. Please avoid using shortcuts to bypass the password-protected files, as it takes away from the fun of solving the challenges and the overall experience. I will never need you to get into a password protected file without using a password. The flag format is gctf{}, some challenges will need to be put into flag format. So if you get an answer of Example_answer the flag will be gctf{Example_answer}. If you need help or hints, feel free to reach out to me via LinkedIn (click LinkedIn logo under my picture to go to my LinkedIn). I'm happy to give you some guidance to keep you moving forward. Also, if you come across any issues or think something is broken (there's a chance that might happen), let me know! I'll make any necessary fixes, re-upload the files, and post updates here. Make sure to check back regularly as you're solving, and keep in mind that if I re-upload any files, you’ll need to reopen them. To avoid losing progress, I recommend storing the flags as you go! Hope everyone enjoys the challenges, and I look forward to hearing your feedback!
